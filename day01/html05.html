입력 양식

항상 form태그 안에 작성할 것. 버튼 클릭시 저장이나 동작 발생.
input type, submit이 일반적인 저장버튼.



form의 데이터 전달방식은 get, post가 가장 많이 사용됨
get 방식은 url로 데이터를 직접 입력해서 전달하는 방식
post 방식은 별도의 방식(숨겨서)으로 데이터를 해당 주소로 전달하는 방식

get 방식은 http://127.0.0.1:5500/day01/html05.html?search=%EB%AF%B8%EC%85%98&date=2025-05-22 로 확인 가능

post 방식은 화면 전환. JavaScript나 서버 기술 필요
